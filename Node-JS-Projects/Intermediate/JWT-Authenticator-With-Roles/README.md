## How to run it? üïπÔ∏è

- Clone the repository
- Run `npm install`
- Create a `.env` file and add the following variables
  - `PORT`
  - `ACCESS_TOKEN_SECRET`
  - `REFRESH_TOKEN_SECRET`
  - `ACCESS_TOKEN_EXPIRY`
  - `REFRESH_TOKEN_EXPIRY`
  - `BCRYPT_SALT`
  - `MONGODB_URI`
  - `CORS_ORIGIN`
- Run `npm run dev` or `npm start`



## Features

- **User Authentication**

  - Register
  - Login
  - Logout
  - Refresh Token

- **User Roles**
  - Admin
  - User
