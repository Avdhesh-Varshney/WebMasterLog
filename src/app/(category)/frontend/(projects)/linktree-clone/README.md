## :zap: Description 📃

<div>
  <p>A Linktree clone project is a simple web app that allows users to create a personalized page with multiple links to their social media profiles, websites, or other resources, all accessible via a single URL.</p>
</div>

<!-- -------------------------------------------------------------------------------------------------------------- -->


## :zap: How to run it? 🕹️

- Clone the repository to your local machine.
- Open the `index.html` file in your web browser to start using the application.
