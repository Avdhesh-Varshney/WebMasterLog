## :zap: Description 📃

<div>
  <p>This project is HTML,CSS and Javascript implementation of a Captcha Generator.It aims to Find the word meaning, part of speech,pronunciation,etc using HTML, CSS, and JavaScript involves building a simple front-end form where users can get captcha, and then using JavaScript actually validate it too.This Showcases the power of HTML,CSS and Javascript.</p>
