## :zap: How to run it? 🕹️

1. Fork the repository.

2. Clone the project.

```
git clone repository-url
```

3. Install dependencies.

```
npm install
```

4. Create and update `.env` file.

```
VITE_GOOGLE_GEMINI_API_KEY=your_api_key
```

5. Run the app.

```
npm run dev
```

6. Start chatting at `localhost:5173`
