## :zap: How to run it? 🕹️

<!-- Add steps how to run this project -->
1. Clone the Repository:

```bash
git clone 
cd React-JS-Projects
cd Typing-Speed-Tracker
```

2. Install Dependencies:
```bash
npm install
```

4. Launch the website:
```bash
npm run dev
```
