## :zap: Description 📃

- A 3D interactive solar system visualization using React and Three.js. It allows users to explore the planets in our solar system, view their relative sizes and positions, and access basic information about each celestial body.
- Custom scaling logic to handle vast size differences in the solar system
- Texture loading and application for realistic planet rendering

## :zap: How to run it? 🕹️

<!-- add the steps how to run the project -->
